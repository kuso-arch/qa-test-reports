## Общее впечатление

Виджет-конструктор для вставки календаря мероприятий на сторонние сайты.  
Основная функция работает, но есть заметные проблемы с валидацией, UX и адаптивностью.

## Найденные баги

| BUG-01  | **High** | Нет валидации размеров виджета               | Задать ширину 50px или 5000px → нажать "Сгенерировать превью" | Показать ошибку или ограничить диапазон (300–1400px) | Превью ломается, виджет становится нечитаемым             | Добавить клиентскую валидацию              |

## 2. Чек-лист тест-кейсов

### Smoke / Happy Path

- [x] Открывается главная страница
- [x] Все шаги отображаются (1–4)
- [x] Можно выбрать тематику
- [x] Можно выбрать одну или несколько стран
- [x] Можно задать ширину и высоту
- [x] Можно выбрать светлую/тёмную тему
- [x] Кнопка "Сгенерировать превью" работает
- [x] Превью отображается
- [x] Кнопка "Скопировать код" работает (код копируется)

### Negative / Edge Cases

- [x] Пустые значения ширины/высоты
- [x] Очень маленькие значения (50px)
- [x] Очень большие значения (5000px)
- [x] Выбор всех стран
- [x] Не выбрана ни одна тематика
- [x] Копирование кода несколько раз подряд

### UX/UI & Адаптивность

- [ ] Адаптив на мобильных (плохо)
- [ ] Удобство использования на планшете
- [ ] Контрастность текста
- [ ] Размер кликабельных областей

## Рекомендации по улучшению

**Высокий приоритет:**
1. Валидация размеров виджета
2. Toast-уведомление при копировании кода
3. Улучшение мобильной версии
4. Live-превью (обновление без нажатия кнопки)

**Средний приоритет:**
- Возможность сохранять и редактировать виджеты
- Больше цветовых пресетов
